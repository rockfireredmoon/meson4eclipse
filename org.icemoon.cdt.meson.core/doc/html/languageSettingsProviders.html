Preprocessor Include Paths, Macros, etc.

Include paths and preprocessor macros for the C/C++ indexer are supplied by Language Settings Providers using the Scanner Discovery mechanism.

The Meson for CDT plugin provides these Settings Providers:

MESON_EXPORT_COMPILE_COMMANDS Parser

This provider instructs Meson to generate a file that contains all the compiler commands-lines which will be executed when a project is build.
The plugin will parse that file and feed any preprocessor include path, macros definition or macros undefine that originates from your MesonLists.txt files to the indexer.
Parsing is done rarely, since its internal cache is invalidated only when the file`s time-stamp is newer, so it should be reasonably fast.
